Shelter Overheating (Summer Failure)

When ambient temperature kills analyzers silently.

Observed Issues

  • Random analyzer trips
  • CPU faults
  • Sensor drift in afternoon

Root Cause

Shelter HVAC undersized for peak summer load, with poor ventilation.

What Was Missed

  • No shelter temperature trend
  • No alarm for HVAC failure

Permanent Fix

  • Upgraded HVAC
  • Added temperature alarm
  • Improved airflow path
Lesson: Shelter failures look like analyzer failures — until you trend temperature.
