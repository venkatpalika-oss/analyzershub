Flash Point Analyzer P-500 — Alarm Codes (Quick Actions)

Practical “first actions” for common P-500 alarms. Use decision trees for step-by-step isolation.

← Back to P-500 Decision Trees →
Technician rule: Fix sample condition + cleanliness + enclosure interlocks first. Many “faults” are caused by contamination, incorrect sample volume, or doors/interlocks not seated.

Ignition / Flame Detection

  • No Ignition Detected — check ignition source (spark/electrode), wiring, and detector cleanliness.
  • Weak/Intermittent Spark — inspect electrode gap, carbon deposits, and HV supply.
  • Flame/Optical Sensor Fault — clean lens/sensor, verify alignment, check cable seating.
No Ignition Tree →

Temperature / Heating

  • Overheating — check heater control, SSR/relay, temperature sensor connection, airflow/venting.
  • Temperature Not Rising — verify heater power, fuse, wiring, controller output.
  • Temp Sensor Error — loose RTD/TC wiring, sensor open/short, moisture ingress.
Overheating Tree →

Test Sequence / Interlocks

  • Test Aborted / Safety Interlock — check enclosure door switch, emergency stop, lid sensor.
  • Sample Not Ready — confirm sample cup seated, correct fill volume, no spillage.
  • Motor/Actuator Fault — check ignition arm / lid actuator movement (if equipped).

Tip: many aborts are from door/lid switches not fully engaged.

Quality / Repeatability

  • Erratic / Non-repeatable Results — sample contamination, dirty cup, unstable ramp rate, drafts.
  • Results Too High/Low — verify method settings, sample volume, reference material validity.
