Decision Tree: Erratic / Non-repeatable Results

Symptom: different flash point values for repeats, poor repeatability, inconsistent trends.

← Back to Trees ← Alarm Codes P-500 Main Page →
Most common causes: contaminated cup, inconsistent sample volume, wrong method parameters, drafts/venting effects.

Step-by-step isolation

1) Verify test method settings:
  • Correct program (open/closed cup mode per your analyzer configuration).
  • Ramp rate and ignition interval match standard practice.
  • Correct sample volume setting (if configurable).
2) Control sample handling:
  • Use the same sample source and minimize evaporation before test.
  • Ensure consistent fill volume each run.
  • Avoid cross-contamination (use clean tools and containers).
3) Cleanliness check (highest impact):
  • Clean and dry the sample cup thoroughly.
  • Inspect lid seal / rim for residue or damage.
  • Clean ignition area and sensor window if vapor deposits exist.
4) Heating stability:
  • Confirm temperature ramp is smooth (no oscillation).
  • Check for drafts/strong airflow affecting heat distribution.
  • Verify temperature sensor reading against external reference if possible.
5) Confirm with a reference material:
  • Run certified flash point reference sample.
  • If reference is stable but product is not → sample variation/handling is the root cause.
  • If reference is unstable too → suspect method settings, heating control, ignition/detection system.
6) Corrective actions:
  • Standardize sample volume and cup cleaning procedure.
  • Repair/replace sensor/ignition components if contamination persists.
  • Service heater control loop if ramp instability is seen.
