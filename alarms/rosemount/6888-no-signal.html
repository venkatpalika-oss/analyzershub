Alarm: No Signal

Manufacturer: Rosemount

Analyzer: 6888 Oxygen Analyzer

Alarm Description

No Signal alarm indicates the analyzer is not receiving oxygen measurement data from the sensor.

Typical Symptoms

  • Oxygen reading absent or zero
  • Calibration fails
  • Heater normal, but no measurement

Probable Causes

  1. Sensor disconnected or damaged
  2. Obstruction in gas flow to sensor
  3. Electronics module failure
  4. Faulty wiring or connections

Field Checks

  • Check sensor connection
  • Inspect gas flow path
  • Check electronics module and wiring
  • Verify sensor condition

Corrective Actions

  • Reconnect or replace sensor
  • Clear obstruction in gas path
  • Repair/replace electronics module if needed
  • Fix wiring/connectors
