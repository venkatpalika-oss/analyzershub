Rosemount 6888 Heater Error - Decision Tree

Step-by-step guide to troubleshoot heater errors:

  1. Measure heater resistance
    • Within spec → Next step
    • Open or shorted → Replace heater element
  2. Check heater power supply voltage
    • Normal → Next step
    • Out of spec → Repair/replace power supply
  3. Inspect wiring and connectors
    • Secure → Check electronics module
    • Loose/damaged → Fix wiring/connectors
  4. Test electronics module
    • Functional → Alarm should clear
    • Faulty → Replace module
