Rosemount 6888 Calibration Failure - Decision Tree

Step-by-step procedure for calibration failure:

  1. Verify calibration gas type and flow
    • Correct → Next step
    • Incorrect → Replace with correct gas
  2. Inspect sensor for contamination
    • Clean → Next step
    • Dirty → Clean or replace sensor
  3. Check electronics module
    • Functional → Alarm should clear
    • Faulty → Replace module
  4. Check wiring/connectors
    • Secure → Alarm cleared
    • Loose/damaged → Repair wiring/connectors
