Rosemount 6888 No Signal - Decision Tree

Stepwise troubleshooting for No Signal alarms:

  1. Check sensor connection
    • Connected → Next step
    • Disconnected → Reconnect sensor
  2. Inspect gas flow to sensor
    • Gas present → Next step
    • Absent → Restore gas flow
  3. Check electronics module
    • Functional → Alarm should clear
    • Faulty → Replace module
  4. Inspect wiring/connectors
    • Secure → Alarm cleared
    • Loose/damaged → Repair wiring/connectors
