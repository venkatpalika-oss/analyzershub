Alarm: High Drift / Unstable Reading

Manufacturer: Rosemount

Analyzer: 6888 Oxygen Analyzer

Alarm Description

High Drift alarm indicates oxygen readings fluctuate beyond acceptable limits. Measurement is unstable.

Typical Symptoms

  • Rapidly changing oxygen readings
  • Calibration fails intermittently
  • Heater and sensor appear normal

Probable Causes

  1. Sensor contamination or aging
  2. Heater instability
  3. Electronics module malfunction
  4. Wiring/connectors loose or damaged

Field Checks

  • Inspect sensor and heater
  • Check electronics module
  • Verify wiring and connections
  • Inspect measurement environment for disturbances

Corrective Actions

  • Clean or replace sensor
  • Repair or replace heater if unstable
  • Repair/replace electronics module
  • Fix wiring/connectors
