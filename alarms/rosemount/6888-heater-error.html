Alarm: Heater Error

Manufacturer: Rosemount

Analyzer: 6888 Oxygen Analyzer

Alarm Description

Heater Error indicates the zirconia sensor heater is not reaching the required temperature. Oxygen measurement may be inaccurate.

Typical Symptoms

  • Oxygen readings unstable or zero
  • Alarm triggers at power-on
  • Calibration cannot complete

Probable Causes

  1. Heater element failure
  2. Power supply issue to heater
  3. Wiring or connection problems
  4. Electronics module malfunction

Field Checks

  • Check heater power supply voltage
  • Inspect wiring and connectors
  • Measure heater resistance
  • Check electronics module status

Corrective Actions

  • Replace heater element if defective
  • Repair/replace heater power supply
  • Fix wiring/connectors as needed
  • Replace electronics module if necessary
