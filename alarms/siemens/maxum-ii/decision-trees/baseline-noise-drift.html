Decision Tree: Baseline Noise / Drift

Symptom: baseline unstable, noisy, drifting, poor integration or false peaks.

← Back to Trees ← Alarm Codes MAXUM II Main →
Rule: If baseline is noisy, do not recalibrate. Fix stability first.

Step-by-step isolation

1) Check carrier supply: correct gas type, sufficient pressure, regulator not hunting, filters not saturated.
2) Leak check: soap test external fittings (carrier/sample), verify column/valve block connections, confirm vent lines are not blocked.
3) Verify oven temperature stability: confirm setpoint reached and stable; check heater/sensor wiring if fluctuating.
4) Confirm detector baseline with carrier only (no sample): if still noisy → detector/flow/temperature issue; if stable → sample contamination/conditioning issue.
5) Sample conditioning check: liquids/condensation, dirty filters, unstable sample pressure/flow, fast loop issues, heat tracing issues.
6) Electrical checks: verify grounding/earthing, check for loose detector signal cables, inspect power supply stability.
7) Corrective actions:
  • Replace carrier filters / purifiers if contamination suspected.
  • Fix leaks and stabilize regulator/flow control.
  • Clean/inspect detector module (per manual) and verify again.
