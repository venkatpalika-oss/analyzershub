Alarm: High Drift / Unstable Reading

Manufacturer: Yokogawa

Analyzer: TDLS8000 Gas Analyzer

Alarm Description

High Drift alarm indicates the TDLS8000 gas readings fluctuate beyond acceptable limits. Measurement is unstable.

Typical Symptoms

  • Rapidly fluctuating gas readings
  • Calibration occasionally fails
  • Alarm triggers intermittently

Probable Causes

  1. Contaminated or aging sensor
  2. Obstruction or instability in measurement path
  3. Laser output instability
  4. Electronics module malfunction

Field Checks

  • Inspect sensor/laser condition
  • Verify measurement path is clean and stable
  • Check laser output stability
  • Inspect electronics module

Corrective Actions

  • Clean or replace sensor/laser
  • Clear and stabilize measurement path
  • Repair or replace laser if needed
  • Replace electronics module if alarm persists
