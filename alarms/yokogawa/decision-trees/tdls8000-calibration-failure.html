TDLS8000 Calibration Failure - Decision Tree

Stepwise procedure to resolve calibration failure alarms:

  1. Verify calibration gas type and flow
    • Correct → Next step
    • Incorrect → Replace with correct gas
  2. Inspect measurement path for contamination
    • Clean → Next step
    • Dirty → Clean or repair path
  3. Check laser output and detector alignment
    • Stable → Next step
    • Unstable → Repair/replace laser or detector
  4. Check electronics module
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
