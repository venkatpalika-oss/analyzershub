OX400 Heater Error - Decision Tree

Follow these steps to diagnose Heater Error alarms:

  1. Check if the analyzer is powered on.
    • Yes → Next step
    • No → Power on analyzer and wait for warm-up
  2. Verify heater status LED.
    • LED normal → Next step
    • LED fault → Inspect heater element and wiring
  3. Measure heater resistance.
    • Within spec → Check wiring and connectors
    • Out of spec → Replace heater element
  4. Inspect heater wiring and connectors.
    • Secure & intact → Check electronics module
    • Loose/Damaged → Repair or replace wiring
  5. Test electronics module.
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
