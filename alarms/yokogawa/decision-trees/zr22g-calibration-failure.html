Decision Tree – ZR22G Calibration Failure

  • Calibration Failure Alarm
    • Is calibration gas available?
      • No → Restore calibration gas supply
      • Yes → Check calibration valve operation
        • Valve not switching → Replace solenoid valve
        • Valve OK → Check gas tubing
          • Leak/blockage → Repair tubing
          • OK → Check sensor response
            • Slow/no response → Replace sensor probe
            • Normal → Review calibration parameters

Field Tip: Expired span gas is a frequent hidden cause.
