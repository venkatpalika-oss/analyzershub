TDLS8000 No Gas Signal - Decision Tree

Step-by-step troubleshooting for No Gas Signal alarms:

  1. Check measurement path for obstruction
    • Clear → Next step
    • Blocked → Clean path
  2. Verify gas reaches analyzer
    • Present → Next step
    • Absent → Restore gas flow
  3. Check laser output and detector
    • Normal → Next step
    • Abnormal → Repair/replace laser or detector
  4. Test electronics module
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
