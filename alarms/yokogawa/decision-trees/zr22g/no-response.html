ZR22G — No Response / Output Frozen (Decision Tree)

Use this when O₂ does not change with process, output stuck, or no update to DCS.

← Back to Yokogawa Trees ZR22G Main Page →

Decision Tree

1) Local display changing?
If local display is changing but DCS is frozen → go to step 6 (output/comms).
2) Heater OK and warmed-up?
If heater not ready → fix heater first (heater error page).
3) Probe plugged / heavy dust?
Plugging causes slow or no response. Inspect/clean protective cap and sensor area.
4) Reference air blocked/leaking?
Bad reference air can cause strange/flat response. Verify filter, line integrity, flow path.
5) Sensor/cell issue?
If stable but never responds to cal gas change → suspect sensor/cell failure or wiring issue.
6) DCS output / scaling / wiring
Check 4–20mA wiring, output range, scaling in DCS, comms mapping (Modbus/Ethernet if used).
7) Still not updating?
Check diagnostics, restart if allowed, inspect terminals for corrosion/water ingress. Escalate if needed.

Fast Checks

  • Compare local reading vs DCS reading
  • Check probe cap/filters for plugging
  • Check heater status and warm-up complete
  • Verify 4–20mA wiring and DCS scaling
  • Check comms status (if digital)
