ZR22G — Heater Error / Temperature Not Ready (Decision Tree)

Use this when heater alarms appear, warm-up never completes, or temperature is unstable.

Safety: Isolate power before opening terminals. Heater circuits can be high temperature/high power.
← Back to Yokogawa Trees ZR22G Main Page →

Decision Tree

1) Power supply OK?
Confirm correct incoming voltage, breakers/fuses OK, no loose terminals.
2) Heater output present?
Check if heater output/SSR/relay is switching. If no switching → suspect controller/SSR/relay output.
3) Wiring/connector condition
Inspect heater wiring, connectors, terminal tightness, signs of overheating/corrosion/water ingress.
4) Heater element continuity
Check heater resistance/continuity (as per service practice). Open circuit → heater element failure.
5) Temperature sensor issue?
If heater is heating but temperature reading abnormal/unstable → suspect temp sensor, wiring, or bad connection.
6) Environmental causes
Extremely cold ambient, damaged insulation, or water ingress can prevent reaching temperature.
7) After repair
Re-energize, allow warm-up completion, then verify stable O₂ reading before calibration.

Common Root Causes

  • Heater element open-circuit
  • Loose/burnt terminals or connector corrosion
  • SSR/relay failure (no heater drive)
  • Temperature sensor wiring fault
  • Water ingress in junction box
