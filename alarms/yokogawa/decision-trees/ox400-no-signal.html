OX400 No Oxygen Signal - Decision Tree

Step-by-step troubleshooting for No Signal alarms:

  1. Check heater operation and warm-up complete.
    • Yes → Next step
    • No → Allow warm-up or fix heater
  2. Verify reference air supply.
    • Present & stable → Next step
    • Absent or blocked → Restore flow, clean filters
  3. Inspect sensor/probe.
    • Output present → Next step
    • No output → Replace sensor/probe
  4. Check sensor wiring to converter.
    • Intact → Check converter electronics
    • Broken → Repair wiring
  5. Check converter electronics module.
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
