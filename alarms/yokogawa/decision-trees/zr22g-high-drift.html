Decision Tree – ZR22G High Drift / Unstable Reading

  • High Drift Alarm
    • Is process condition stable?
      • No → Stabilize process first
      • Yes → Inspect probe for contamination
        • Dirty/corroded → Clean or replace probe
        • Clean → Check reference air leakage
          • Leak present → Repair tubing
          • No leak → Check grounding and shielding
            • Poor grounding → Correct earthing
            • Grounding OK → Sensor end-of-life → Replace sensor

Field Tip: Ash buildup is the #1 cause in boilers and kilns.
