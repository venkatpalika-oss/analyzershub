TDLS8000 High Drift / Unstable Reading - Decision Tree

Step-by-step diagnosis to resolve high drift alarms:

  1. Inspect sensor/laser for contamination or aging
    • Good → Next step
    • Dirty/worn → Clean or replace
  2. Check measurement path stability
    • Stable → Next step
    • Unstable → Stabilize path
  3. Verify laser output stability
    • Stable → Next step
    • Fluctuating → Repair/replace laser
  4. Check electronics module
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
