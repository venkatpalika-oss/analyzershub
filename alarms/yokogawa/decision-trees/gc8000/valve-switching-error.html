GC8000 — Valve Switching / Injection Error (Decision Tree)

Use when valves fail to switch, timing errors occur, or injections are inconsistent.

Golden rule: A valve switching error can be mechanical, pneumatic, or electrical. Check actuation supply and timing first.
← Back to Yokogawa Trees GC8000 Main Page →

Decision Tree

1) Actuation supply OK?
If pneumatic valves used: check instrument air pressure/quality. If electric: check supply voltage.
2) Valve command occurs?
Confirm controller is issuing the switch command. Check timing/method step sequence.
3) Mechanical sticking?
If valve sticks: contamination, wear, moisture. Inspect, clean, replace seals as per maintenance.
4) Solenoid/driver issue
For pneumatic valves: solenoid coil/driver failure. Check coil, wiring, output status.
5) Sample loop filling problem
If injection inconsistent: check sample pressure, restriction, loop blockage, sample conditioning.
6) After repair
Run multiple cycles and confirm repeatability and stable chromatogram peaks.

Fast Checks

  • Instrument air pressure & dryness (if pneumatic)
  • Solenoid coil and wiring
  • Valve actuation response (listen/feel switching)
  • Sample pressure/flow and loop restrictions
