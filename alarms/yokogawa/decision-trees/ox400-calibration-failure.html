OX400 Calibration Failure - Decision Tree

Stepwise procedure to resolve calibration failure alarms:

  1. Verify reference gas type and purity.
    • Correct → Next step
    • Incorrect → Replace with correct gas
  2. Check gas flow to analyzer.
    • Stable flow → Next step
    • Obstructed → Restore flow, clean tubing
  3. Check heater stability.
    • Stable → Next step
    • Unstable → Repair heater
  4. Inspect sensor/probe condition.
    • Clean → Next step
    • Contaminated → Clean or replace sensor/probe
  5. Check wiring and electronics.
    • Functional → Alarm should clear
    • Faulty → Replace electronics module
