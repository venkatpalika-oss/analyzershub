Yokogawa ZR22G — Alarm Codes (Quick Actions)

Practical first actions for common ZR22G alarms. Use decision trees for step-by-step isolation.

← Back to ZR22G Decision Trees → Yokogawa Trees Hub →
Tip: Most ZR22G issues are from heater instability, reference air problems, or probe fouling. Fix stability first — then calibrate.

Heater / Temperature Alarms

  • Heater Error / Temp Not Ready — check power, wiring, heater continuity, temp sensor wiring.
  • Warm-up Not Completing — verify supply voltage, SSR/relay output, insulation/water ingress.
Heater Error Tree →

Calibration Alarms

  • Calibration Failure — verify correct gas, flow, no leaks in cal tubing/manifold, adequate stabilization time.
  • Span Not Reaching — most common: cal gas leak/low flow, probe plugging, unstable temperature.
Calibration Tree →

Drift / Unstable Reading

  • High Drift — check reference air filter/line leakage, probe contamination, heater stability.
  • Unstable O₂ — check grounding, moisture ingress, terminal tightness, process swings.
High Drift Tree →

No Response / Frozen Output

  • No Response — check probe plugging, heater not ready, reference air blockage/leak.
  • DCS Frozen Only — check 4–20 mA wiring/scaling or digital comms mapping.
