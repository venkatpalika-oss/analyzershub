Alarm: No Oxygen Signal

Manufacturer: Yokogawa

Analyzer: OX400 Oxygen Analyzer

Alarm Description

The analyzer is not receiving a valid oxygen measurement signal. Output may be frozen, zero, or invalid.

Typical Symptoms

  • Oxygen value fixed at 0%, 21%, or last known value
  • Analyzer shows sensor error or invalid reading
  • Calibration cannot be performed

Probable Causes

  1. Heater not operating or warm-up incomplete
  2. Reference air supply blocked or missing
  3. Sensor probe failure
  4. Signal wiring fault
  5. Converter electronics failure

Field Checks

  • Verify heater operation and stable temperature
  • Check reference air flow and filters
  • Inspect probe for physical damage
  • Measure sensor output at converter terminals
  • Check probe-to-converter wiring continuity

Corrective Actions

  • Allow full warm-up or restore heater operation
  • Restore reference air supply and clean filters
  • Replace sensor probe if no output
  • Repair or replace damaged wiring
  • Replace converter electronics if required

Impact on Measurement

Analyzer output must not be used until normal operation is restored.
