ABB GC Alarm Codes (Shared)

Carrier / Pressure / Flow

Symptoms: no peaks, unstable baseline, retention shifts.

First checks: carrier cylinder, regulator stability, leaks, pressure hunting.

Oven / Temperature

Symptoms: oven not ready, heater fault, drift/repeatability issues.

First checks: setpoint vs actual, heater power, sensor integrity, airflow.

Valve / Sequence

Symptoms: injection failure, stuck valve, timing faults.

First checks: instrument air, solenoids, valve actuation, method timing.

Communication / DCS

Symptoms: stale values, bad quality, missing updates.

First checks: IP/network, mapping, protocol service, update cycle timing.
