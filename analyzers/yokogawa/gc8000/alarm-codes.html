Alarm Code Table

Alarm / Message Type Severity Typical Symptoms Likely Causes Fast Checks (5–10 min) Next Actions
Carrier Low / Pressure Low CRITICAL No peaks, unstable baseline, run fails early Empty cylinder, regulator creep/hunting, leak, blocked restrictor/filter Check cylinder pressure, regulator outlet, leak test quick fittings Replace cylinder/regulator, repair leaks, clean/replace filters; then run verification
Flow Unstable MAJOR Retention time shift, poor repeatability Regulator hunting, micro-leaks, temperature instability, sample restriction Observe outlet pressure stability; check fittings near manifold Stabilize carrier + oven; fix leaks; verify 2–3 runs stable
Oven Not Ready MAJOR Runs blocked or results flagged invalid Heater fault, fuse, sensor issue, ventilation/cooling issue Compare setpoint vs actual; check cabinet airflow Restore temperature stability; replace heater/sensor per procedure
Heater Fault / Heater Overload CRITICAL No warm-up / temp drops, or temp runaway Heater open circuit, SSR/relay fault, wiring issues Check power/fuse status; inspect terminals for burn Electrical maintenance: replace heater/SSR; verify stable hold
Temperature Sensor Error CRITICAL Temp reading jumps, oven control unstable Loose sensor wiring, failed RTD/TC, grounding/noise Reseat sensor connectors; check for damaged cable Replace sensor; confirm stability for 15–20 min, then run test
Valve Not Switching / Solenoid Fault MAJOR No peaks / wrong peaks, timing issues Low instrument air, wet/dirty air, solenoid coil fault, valve sticking Check air pressure/quality; observe switching; check solenoid LED Fix air quality, swap solenoid if possible, service/replace valve
Sequence Error / Method Error MAJOR Run aborts, abnormal cycle behavior Incorrect timing/config changes, module mismatch after maintenance Confirm correct method loaded; compare with last known good Restore backup method; re-verify with standard/known stream
No Peaks / Weak Peaks MAJOR Flat chromatogram or tiny peaks Sample not reaching, blocked filter, leak, injection failure, carrier issue Check sample pressure/flow, filters, carrier supply Use sampling guide + carrier/valve tree to isolate and correct
Baseline Noise / Instability MINOR Noisy baseline, integration errors Pressure instability, contamination, oven fluctuations Check carrier stability + oven stability Stabilize utilities; inspect for contamination; verify repeatability
Communication Fail / Link Down MAJOR DCS stale values, bad quality flags Network cable/switch issue, IP mismatch, firewall/protocol service down Check link LEDs, IP/subnet/gateway Restart service (if allowed), correct mapping/registers, confirm cycle update timing
Bad Quality / Data Stale MINOR DCS value not updating every scan Update rate mismatch (GC cycles), tag mapping/scaling, quality flag mapping Compare local value vs DCS; check update timestamp per run Align DCS scan to cycle; fix mapping/scaling; validate historian timestamps
Best practice: Always verify fixes with 2–3 stable consecutive runs before handing over results to operations.
