Yokogawa ZR22G — Zirconia Oxygen Analyzer

In-situ zirconia O₂ analyzer used for combustion control (boilers, heaters, furnaces). Technician focus: probe condition, heater stability, reference air, calibration gas handling, and drift control.

← Back to Search Alarm Decision Trees → Alarm Codes →
Technician rule: If O₂ is unstable or drifting, check probe condition + reference air + heater stability first. Don’t calibrate until the reading is stable.

Working Principle

ZR22G uses a zirconia sensor (solid electrolyte) that generates an electrical signal based on the difference in oxygen concentration between process gas and reference air.

  • Sensor output changes with oxygen partial pressure difference (process vs reference).
  • Heater maintains sensor temperature for stable, correct measurement.
  • Accuracy depends on clean probe, stable temperature, proper reference air, and correct calibration gas.

Main Components (Field Focus)

  • Probe / sensor assembly: zirconia cell, protective filter/cap, mounting hardware.
  • Heater & temperature control: heater element, temperature sensor, controller.
  • Reference air system: reference air line, filter, flow control (clean/dry air).
  • Calibration system: calibration gas connection, solenoid/valves (if equipped), tubing.
  • Converter/transmitter: display/HMI, outputs, diagnostics, DCS communication.

Calibration Procedure (Technician Sequence)

1) Warm-up complete: confirm heater OK and reading stable.
2) Verify reference air: clean, dry, flowing (no blockage/leaks).
3) Check probe/filter condition: no heavy dust buildup or plugging.
4) Apply Zero/Low O₂ calibration gas (as per plant practice). Wait stabilization.
5) Apply Span calibration gas. Wait stabilization. Adjust span value if required.
6) Return to process. Verify stable reading and output to DCS.

Tip: If you see unstable O₂ during calibration, fix probe/heater/reference air first.

Major Troubleshooting (Top Issues + Fast Checks)

  • High drift / unstable O₂: probe contamination, reference air leak/blockage, heater instability.
  • Slow response: plugged filter/cap, heavy dust, long sample diffusion time due to blockage.
  • Heater error / low temp: heater element failure, wiring, controller, power supply.
  • Calibration failure: wrong gas, leaks in cal line, insufficient stabilization time, unstable sensor temp.
  • DCS mismatch: scaling, range, output wiring, comms mapping.
