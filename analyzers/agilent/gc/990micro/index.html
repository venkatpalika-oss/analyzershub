Agilent 990 Micro GC

Portable micro gas chromatograph for fast, on-site gas composition analysis in refineries, gas plants, pipelines, and laboratories.

Overview (Field Perspective)

The Agilent 990 Micro GC is a compact, modular gas chromatograph using micro-machined columns and Thermal Conductivity Detectors (TCDs). It is designed for rapid, portable gas analysis rather than continuous online process control.

Technician rule: The 990 Micro GC is extremely sensitive to sample pressure stability, moisture, and contamination. Always condition and regulate the sample before connection.

Working Principle

The 990 Micro GC separates gas components using heated micro-columns. A TCD detects changes in thermal conductivity as components elute.

  • Injection: Fast micro-valve injection.
  • Separation: Micro-columns with short analysis time.
  • Detection: TCD converts thermal changes to signal.
  • Calculation: Concentration calculated from calibration data.

Main Components

  • Micro GC column modules (application-specific)
  • TCD detectors (integrated per module)
  • Internal carrier gas system
  • Sample pressure regulator / restrictor
  • Embedded controller & PC software
  • Ethernet / USB communication

Calibration Procedure

1) Power up and allow full thermal stabilization.
2) Verify stable sample pressure (critical).
3) Apply certified calibration gas.
4) Calibrate each active module individually.
5) Verify response with a second standard.

Tip: Do not calibrate if baseline noise or pressure fluctuation exists.

Major Troubleshooting Areas

  • No peaks: Insufficient inlet pressure, blocked inlet.
  • Poor repeatability: Pressure fluctuation, moisture ingress.
  • Low sensitivity: Contaminated detector or aging module.
  • Retention time shift: Module temperature instability.
  • Communication loss: Network or software configuration issue.
