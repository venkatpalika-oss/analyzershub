1) Startup & Health Checks

A — Visual inspection
  • Check cabinet condition: dust, moisture, loose tubing, abnormal odor/heat.
  • Confirm utilities: stable power, instrument air (if used), carrier gas, sample supply.
  • Check sample conditioning: filters, drains, traps, heat tracing (if used).
B — Status & alarms
  • Open local interface/software and check current alarms (record alarm text + time).
  • Confirm analyzer is in RUN and method/schedule is active.
  • Confirm last analysis timestamp is updating normally.
C — Quick checks (most common causes)
  • Carrier supply pressure OK and stable.
  • Sample pressure/flow OK (no restriction/plugging).
  • Oven temperature at setpoint and stable.
Healthy system signs: stable pressures, stable oven temperature, repeatable runs, no active critical alarms.

2) No Update / No Results / Stale Values

Symptoms
  • DCS shows stale values (no update), bad quality, or zeros.
  • Local GC run not happening (no new timestamp) or results not calculated.
Check 1 — Is the GC running locally?
  • If not running: confirm schedule/method start, check interlocks, check critical alarms.
  • If running: go to Check 2.
Check 2 — Are chromatograms/peaks present?
  • No peaks: likely sample not reaching / injection not happening / carrier problem → see sections 5–8.
  • Peaks present but no result: suspect method/calculation/config or comms mapping → see section 9.
Check 3 — DCS link vs analyzer link
  • If values update locally but not in DCS → comms/protocol/mapping issue.
  • If nothing updates locally either → GC run path or sampling/carrier/oven/valve issue.
If urgent: record (1) last good timestamp, (2) alarm list, (3) sample & carrier pressures, (4) oven temp, and escalate.

3) Bad Quality / Unstable CV–Wobbe

Symptoms
  • CV/Wobbe fluctuates more than normal.
  • Repeatability poor between runs.
  • Sudden step change without process reason.
Most common causes
  • Sample conditioning issue (liquid carryover, water, heavy HC contamination).
  • Unstable sample pressure or flow restrictions.
  • Carrier pressure instability or small leaks.
  • Oven temperature instability.
  • Valve timing/sequence inconsistency.
Fix actions
  • Drain traps, check coalescing filters, replace blocked sample filters.
  • Verify sample pressure regulator stability (no hunting).
  • Leak test around valve manifold and fittings.
  • Confirm oven at setpoint and stable for 15–20 min.

4) Drift / Repeatability Issues

Quick diagnosis
  • Drift slowly over hours/days: contamination, regulator creep, aging components.
  • Drift suddenly: leaks, sample upset, method/config change, valve issue.
Checks
  • Review trend: CV/Wobbe vs sample pressure vs carrier pressure vs oven temperature.
  • Run 2–3 repeat analyses on same stream and compare.
  • Verify calibration gas cylinder and regulator condition (no creep).
Fix actions
  • Replace suspect regulators; leak test calibration line and switching valves.
  • Service sample conditioning to remove contamination source.
  • Restore method from last known good backup if configuration changed.

5) Sample System Troubleshooting

Symptoms
  • No peaks / weak peaks / slow response.
  • Frequent filter plugging or liquid in traps.
  • Pressure alarms on sample side.
Checks
  • Check sample filters/coalescers and replace if dirty.
  • Drain liquid traps and verify heat tracing (if used).
  • Check sample regulator and needle valve/restrictor for blockage.
  • Confirm sample supply pressure from process is available.
Verification: stable sample pressure/flow, consistent peak response, no new liquid accumulation.

6) Carrier / Pressure / Flow Issues

Symptoms
  • No peaks, unstable retention, noisy baseline, “carrier low” type alarms.
Checks
  • Confirm cylinder pressure and regulator outlet setpoint.
  • Check for regulator creep or oscillation (hunting).
  • Leak test carrier path, valve block, detector fittings.
  • Inspect filters/restrictors for blockage.
Fix actions
  • Replace empty cylinder; open valves fully.
  • Replace faulty regulator; repair leaks; replace ferrules.
  • Replace clogged filters/restrictors and recheck stability.

7) Oven / Temperature Issues

Symptoms
  • Oven not ready, slow ramp, unstable temperature, heater/sensor alarms.
Checks
  • Compare setpoint vs actual temperature; look for oscillations.
  • Check door seal/gasket and cabinet airflow for heat loss.
  • Check heater power status and sensor readings (open/short indications).
Fix actions
  • Correct enclosure sealing/airflow issues.
  • Repair wiring, replace faulty sensor/heater as per plant procedure.
  • Confirm stable temperature for 15–20 minutes then repeat run.

8) Valve / Sequence Issues

Symptoms
  • No injection events, wrong peak order, retention shift, valves not switching.
Checks
  • Confirm pneumatic supply (instrument air) and regulator.
  • Listen/observe valve switching during a run.
  • Check solenoid status/diagnostics (command vs actual).
  • Leak test valve manifold and check for contamination in pneumatic lines.
Fix actions
  • Restore air supply; drain water in air lines; replace sticky solenoids.
  • Verify method timing vs last known good backup.
  • Service/replace leaking or stuck valves if suspected.

9) DCS/Network/Communication Issues

Symptoms
  • DCS shows bad quality, stale values, or no update.
  • Analyzer shows values locally but not in DCS.
Checks
  • Verify network link (cable/switch LED), IP/subnet/gateway.
  • Confirm protocol status (Modbus/OPC/etc.) is running.
  • Check mapping/tag list and scaling (especially CV/Wobbe units).
  • Look for time sync issues if historian uses timestamps.
Fix actions
  • Correct IP settings and eliminate duplicate IP conflicts.
  • Restart comm service (if allowed) and verify tag mapping with DCS team.
  • Confirm quality flags and update rate match DCS expectations.

10) Preventive Maintenance

  • Weekly: check sample filters/traps, drain liquids, verify pressures stable.
  • Monthly: leak check critical fittings, review drift/quality trends, verify calibration.
  • Quarterly: verify valve actuation, service sample conditioning, check oven stability.
  • Planned: maintain regulators, solenoids, and replace worn components proactively.

11) Escalation Checklist (Before Vendor Support)

Collect these details
  • Analyzer model + serial (and module IDs if shown)
  • Alarm text + timestamp + screenshots
  • Last good run time + last calibration time
  • Sample pressure/flow, carrier pressure, oven temperature (actual vs setpoint)
  • What you already tried (filters replaced, leak test, restart, etc.)
