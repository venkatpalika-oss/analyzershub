Valve / Sequence Alarm

Decision tree for “Valve not switching”, “Sequence fault”, “Injection timing error”, “No peaks due to valve”.

Step 1 — Confirm Symptom Type

Identify which case matches:
  • Case A: No peaks / very weak peaks
  • Case B: Peaks present but wrong order/retention shift
  • Case C: Runs fail early or sequence stops

Step 2 — Pneumatic Supply (Most Common)

Check:
  • Instrument air present and stable
  • No water/oil contamination in air line
  • Regulator set correctly for valve manifold
If air supply is low/dirty → valves may not actuate fully → clean/dry air and re-test.

Step 3 — Observe Valve Switching During a Run

  • Start a run and observe: do you hear/see switching at expected times?
  • Check diagnostics: command vs actual state (if available)
  • Confirm solenoid LEDs (if present) toggle when commanded

Step 4 — Solenoid / Coil / Electrical

If valve does not move:
  • Check solenoid coil connection and tightness
  • Swap with a known good solenoid channel (if possible) to isolate
  • Check for blown fuse/IO fault
If swapping solenoid fixes issue → replace faulty solenoid/coil.

Step 5 — Valve Block Restriction / Sticking

If solenoid works but switching still unreliable:
  • Contamination in valve (oil, sticky residue)
  • Mechanical wear / internal leakage
  • Incorrect pressure on valve actuator
Actions:
  • Check air quality and drain air filter
  • Service or replace sticking valve as per OEM procedures
  • After replacement: run repeat test for stable retention and consistent peaks

Step 6 — Method/Timing Configuration Check

If valves switch correctly but results wrong:
  • Compare method timing with last known good backup
  • Check if recent changes were made (after maintenance or software update)
  • Confirm correct stream selection and calibration sequence
Wrong timing can look like “hardware fault” but is actually configuration.

Verification

Pass criteria:
  • Valve events occur consistently every run
  • Peaks appear normal and repeatable
  • No sequence faults during full cycle
