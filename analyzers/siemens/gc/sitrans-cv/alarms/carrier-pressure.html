Carrier / Pressure / Flow Alarm

Decision tree for “Low carrier”, “No peaks”, “Injection failure” alarms.

Step 1 — Check Carrier Cylinder

• Verify cylinder is not empty • Main valve fully open • Correct carrier gas type
If cylinder pressure is zero → replace cylinder and leak test

Step 2 — Regulator & Pressure Stability

• Check outlet pressure matches SITRANS CV specification • Observe pressure for hunting or drop
Unstable pressure = faulty regulator → replace

Step 3 — Leak Check

• Leak test fittings from regulator to valve manifold • Check ferrules and tubing bends
Any leak → repair and re-test before restarting GC

Step 4 — Injection & Flow Path

• Confirm valves switch during run • Check restrictors / filters for blockage

Result

Carrier pressure stable, peaks return, alarm clears automatically after next successful run
