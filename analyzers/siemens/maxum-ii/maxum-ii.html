Siemens MAXUM Edition II — Process Gas Chromatograph

Online process GC for continuous gas composition monitoring. Technician focus: sampling quality, carrier gas stability, valve switching, oven stability, detector stability, and DCS communications.

← Back to Search Alarm Decision Trees → Chromatogram Guide →
Technician rule: Always fix sample conditioning + carrier pressure/flow stability first. Most GC “faults” are caused by wet/dirty sample, pressure hunting, leaks, or valve timing issues.

Working Principle

The GC separates gas components in one or more columns and measures each component using a detector. Each analysis cycle typically runs: inject sample → separate → detect peaks → calculate composition.

  • Separation: components travel at different speeds through the column(s).
  • Detection: detector creates a signal producing peaks (chromatogram).
  • Calculation: peak area → concentration using calibration factors/response factors.

Accuracy depends on: stable carrier flow, stable oven temperature, clean sample, and consistent valve switching.

Main Components (Field Focus)

  • Sample system: sample probe/tap, filters/coalescers, regulators, restrictors, drains.
  • Carrier gas system: carrier supply/regulators, traps/filters (if used), leak-tight fittings.
  • Valves & sample loop: injection valve(s), switching valve(s), loop volume, timing.
  • Oven & temperature control: heater, temp sensor, controller (stability = stable retention time).
  • Columns: column module(s), restrictions, fittings.
  • Detector(s): detector module + electronics (noise/instability affects peak area).
  • Controller & comms: HMI/controller, alarms/status, network/OPC/Modbus mapping.

Calibration Procedure (Technician Sequence)

1) Confirm stable operation: no sample/carrier/oven alarms; stable cycle time.
2) Verify carrier pressure/flow is stable (no hunting). Leak check if instability exists.
3) Verify sampling is dry/clean (filters, drains, no liquid carryover).
4) Apply certified calibration standard. Allow 2–3 stable runs.
5) Update response factors/calibration factors per method; save and document.
6) Run verification standard. Confirm % error is within acceptance criteria.
7) Return to process. Confirm stable chromatogram and DCS values/quality.

Tip: If retention time drifts or peak area is unstable, stop calibration and fix carrier/oven/sample issues first.

Major Troubleshooting (Top Issues + Fast Checks)

  • Retention time shift: oven temperature drift, carrier flow/pressure change, column aging.
  • Peak area unstable / noisy baseline: carrier instability, leaks, contamination, detector noise.
  • Missing peaks / low response: injection valve/loop issue, restriction, sample not reaching GC, detector fault.
  • Repeatability poor: valve timing, pressure hunting, wet sample, filter saturation.
  • Results wrong only in DCS: mapping/scaling, wrong register/point, quality bit handling.
