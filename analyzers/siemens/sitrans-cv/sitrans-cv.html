Siemens SITRANS CV — Calorific Value / Wobbe Analyzer

System for Calorific Value and Wobbe Index measurement for fuel-gas applications. Technician focus: stable sampling, pressure/flow stability, moisture control, stable calculations, and DCS communications.

← Back to Search Alarm Decision Trees → Chromatogram Guide →
Technician rule: If CV/Wobbe is unstable, do NOT calibrate first. Stabilize sample condition + pressure/flow, then verify outputs and DCS scaling.

Working Principle

SITRANS CV provides fuel quality indication used in combustion control. The output relates to energy content and specific gravity, and the system relies heavily on stable sampling and pressure.

Concept: Wobbe Index = Heating Value / √(Specific Gravity)

  • Stable sample is critical — moisture or liquid carryover causes major errors.
  • Stable pressure/flow is critical — regulator hunting causes unstable output.
  • DCS mapping must include correct scaling and quality/status bits.

Main Components (Field Focus)

  • Sampling system: probe/tap, tubing, isolation valves, leak-tight fittings.
  • Conditioning: filters/coalescers, moisture control, drains, heat tracing (if used).
  • Pressure/flow control: regulators, restrictors, flow indication.
  • Measurement/calculation module: measurement section + internal computation.
  • Controller & comms: network interface, outputs to DCS (OPC/Modbus/etc.).

Calibration Procedure (Technician Sequence)

1) Verify sampling is stable: dry gas, no liquid carryover, filters/drains OK.
2) Verify pressure/flow is stable: no regulator hunting, no restrictions/leaks.
3) Stabilize reading locally first (trend steady for sufficient time).
4) Apply reference/standard as per site procedure (zero/verification).
5) Apply span/verification point (if applicable) and confirm stable output.
6) Return to process and confirm stable CV/Wobbe and DCS values/quality.

Tip: If output is fluctuating, calibration will fail or won’t hold—fix sample/pressure first.

Major Troubleshooting (Top Issues + Fast Checks)

  • Output fluctuates: regulator hunting, wet sample, blocked drains, filter saturation, leaks.
  • Output suddenly steps: process composition change or liquid carryover during upset.
  • Wrong vs lab: compare only after stable period; verify scaling/mapping in DCS.
  • Calibration fails: reference flow/pressure wrong, sample unstable, leaks or moisture.
  • DCS issues: stale values, wrong scaling, bad quality bit mapping, polling mismatch.
