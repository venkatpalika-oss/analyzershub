Calibration Procedure (Field)

1) Ensure analyzer is warmed-up and stable (no active sample faults).
2) Verify sample flow/pressure and conditioning OK.
3) Apply Zero gas (or reference condition). Allow stabilization.
4) Apply Span gas. Allow stabilization. Adjust span value.
5) Return to process sample. Verify reading and record results.
6) Update calibration log + check output to DCS.

Major Troubleshooting (Top Issues)

  • Unstable reading / drift: check sample conditioning, leaks, temperature stability
  • No reading / low reading: check flow, blockage, sensor status, alarms
  • Calibration fails: check calibration gas, flow, leaks, sensor aging
  • DCS not updating: check comms, mapping, scaling, quality status

Rule: Fix sample + utilities first, then sensor/electronics last.
