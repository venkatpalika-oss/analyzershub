Decision Tree: Oven / Temperature

← Back to Alarms Hub ABB GC Codes → Chromatogram Guide
Goal: Stabilize oven temperature before accepting portable results.
1) Check setpoint versus actual temperature.
2) If temperature is low: verify power supply and heater path.
3) If temperature hunts: check sensor integrity, wiring, and airflow.
4) Allow sufficient stabilization time.
5) Run 2–3 cycles and confirm repeatable results.
