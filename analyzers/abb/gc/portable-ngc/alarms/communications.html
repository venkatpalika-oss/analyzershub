Communication / Export Alarm (Portable NGC)

1) Cable / driver / port
Confirm correct cable and connection method. Try different USB/Ethernet port.
2) Network settings
Confirm IP/subnet if Ethernet is used. Avoid duplicate IP.
3) Export behavior
Export occurs after run completion. Confirm run finished successfully.
If run fails, export may be incomplete or blocked.
Verify: export file/time updates after each completed cycle.
