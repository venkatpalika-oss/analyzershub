Decision Tree: Oven / Temperature

← Back to Alarms Hub ABB GC Codes → Chromatogram Guide
Goal: get oven stable at setpoint before judging retention time or repeatability.
1) Compare setpoint vs actual. Note alarms and timestamps.
2) If actual never reaches setpoint: check heater power path, fuses, relays, wiring.
3) If temperature hunts: check sensor integrity, loose connections, airflow/venting.
4) Confirm cabinet ventilation and ambient temperature are acceptable.
5) Run 2–3 cycles and confirm stable retention time.
