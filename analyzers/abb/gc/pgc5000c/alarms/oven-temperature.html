Oven / Temperature Alarm (PGC5000C)

1) Setpoint vs actual
Confirm actual temperature reaches and holds setpoint steadily.
Temp instability = retention shift + poor repeatability.
2) Check heater power path
Check fuses/breakers, heater power status, loose terminals.
3) Sensor / wiring
Inspect RTD/TC wiring, connectors, and any recent maintenance disturbance.
4) Cabinet cooling/ventilation
Ensure ventilation is not blocked; check fan operation if applicable.
Verify: hold stable for 15–20 minutes then run test.
