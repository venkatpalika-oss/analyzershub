Decision Tree: Oven / Temperature

← Back to Alarms Hub ABB GC Codes → Chromatogram Guide
Goal: stabilize oven temperature at setpoint before judging sulfur results.
1) Check setpoint vs actual, note alarms.
2) If low: check heater power path/fuses/relays.
3) If hunting: check sensor and wiring, airflow.
4) Confirm cabinet ventilation & ambient conditions.
5) Run 2–3 cycles and verify stable retention time.
