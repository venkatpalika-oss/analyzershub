ABB AZ25 – Alarm Decision Trees

Alarm troubleshooting guidance for ABB AZ25 TDLAS analyzers. These alarms are commonly related to optical path condition, signal strength, alignment, purge flow, and process disturbances.

Technician note: AZ25 alarms are often caused by beam attenuation, window fouling, purge issues, or process upsets. Always verify the optical path before replacing analyzer hardware.

Common ABB AZ25 Alarms

  • No Signal / Beam Lost
  • Low Signal Strength / Attenuation
  • Calibration Fault
  • Purge / Reference Path Fault
