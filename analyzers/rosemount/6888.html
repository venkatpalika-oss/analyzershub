Rosemount 6888 Zirconia Oxygen Analyzer

Analyzer Overview

The Rosemount 6888 is a Zirconia-based oxygen analyzer widely used in CEMS and combustion control applications. It provides accurate and reliable oxygen measurements in industrial processes.

Key Features

  • Zirconia sensor for oxygen measurement
  • Fast response and high accuracy
  • Stable operation in harsh industrial environments
  • Supports field calibration and diagnostics

Typical Applications

  • Boilers and furnaces
  • Continuous Emissions Monitoring Systems (CEMS)
  • Combustion control and optimization
  • Industrial process monitoring

Common Alarms & Troubleshooting

Alarm Description Diagnosis
Heater Error Heater not reaching set temperature Start Diagnosis
No Signal Oxygen reading missing Start Diagnosis
Calibration Failure Zero/span calibration failed Start Diagnosis
High Drift / Unstable Reading Oxygen readings fluctuate beyond limits Start Diagnosis

Maintenance & Calibration Notes

  • Ensure heater reaches stable temperature before measurement
  • Perform zero/span calibration with certified gas
  • Check wiring and electronics periodically
  • Inspect zirconia sensor for contamination or wear
