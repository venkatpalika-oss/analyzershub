How to use this hub (fast)

  1. Confirm the symptom using trend + alarms + operator feedback
  2. Check utilities (power, instrument air, temps, flow)
  3. Check sample system (probe → line → conditioning → flow)
  4. Check calibration path (gas, regulator, manifold, valves)
  5. Verify recovery with stable trend + repeat check
Most analyzer faults are caused by sample system problems, condensation, leaks, or utility instability — not electronics.

Start by Symptom

Reading Low / Zero

Plugging, restriction, wrong range, wrong gas, or analyzer not warmed up.

  • Verify flow at analyzer inlet
  • Check filters/DP and probe blockage
  • Confirm correct range and scaling

Reading High

Leaks (air ingress), process upset, wrong span value, drift.

  • Leak test manifolds and sample line
  • Compare with calibration gas response
  • Check process condition changes

Unstable / Noisy Trend

Pressure fluctuations, moisture, unstable regulator, failing pump.

  • Check pressure/flow stability
  • Inspect moisture carryover
  • Check grounding / signal wiring

Slow Response Time

Restriction, long lines, low flow, condensation, tired pump.

  • Inspect filters and restriction
  • Check pump health and flow
  • Confirm heater temperatures

Calibration Failure

Wrong gas, leaking manifold, low flow, unstable regulator, warm-up not complete.

  • Verify cylinder concentration + expiry
  • Leak test calibration manifold
  • Confirm stable switching of solenoids

Communication / Data Gaps

Network, mapping, DAHS tag issues, power resets, incorrect scaling.

  • Check comm status + IP/port
  • Verify tag mapping and scaling
  • Check timestamps / data flags
